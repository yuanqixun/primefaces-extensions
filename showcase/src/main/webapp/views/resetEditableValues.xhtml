ResetEditableValues is a component which can be applied to any UICommand component like CommandLink, CommandButton, MenuItem, RemoteCommand and HotKey. This component enables to reset editable components derived from EditableValueHolder. This can be used to restore default values or reset validation state.

Example: if you clear a form, you usually set current backing bean values to default values and update the form elements. The problem is, if validation was already triggered before and some input fields were invalid. The validation state remains and survives further updates. You will see then last submitted values and / or red borders surrounding PrimeFaces' form elements. Use pe:resetEditableValues in this case.
